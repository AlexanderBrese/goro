# About

Goro is yet another pomodoro tracker that tries to gamify the process.

Gamified Elements are:

  • Progression through levels
  • Unlockable sounds
  • Daily Goals
  • Leaderboard

# Usage

Just start a new pomodoro, type in a new task you want to work on and start the pomodoro. You can optionally get some more insight into your daily habits by looking at the stats.


# Privacy

Your data is being stored locally thus it is private. Note however that for technical reasons you have to upload your data whenever you want to access the leaderboard or view elaborate stats that are being calculated and learned on the cloud.
